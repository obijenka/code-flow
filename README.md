# CodeFlow - Modern Landing Page

A responsive landing page built with React.

## Tech Stack

- React 19
- Tailwind CSS
- Vite 
- Lucide React (icons)
- React Syntax Highlighter

## Features

- Fully responsive design
- Gradient effects
- Mobile-friendly menu
- Backdrop blur effects (glassmorphism)
- Interactive code blocks with syntax highlighting
- Modern UI components
